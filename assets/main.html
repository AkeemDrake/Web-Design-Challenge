Main Objective:

The main objective of this project: compare the weather of different cities across the world by comparing the distance of the cities from the equator.

The different weather types that were looked at were:

  • Humidity (%)
  • Cloudiness (%)
  • Temperature (F)
  • Windspeed (mph)
Some of the applications used in this project were:
  • API's
  • Python
  • Jupyter Notebook
